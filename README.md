## 🐶 서비스 소개
**QuantumGuinea**는 반려동물과 함께 여행을 계획하는 사람들을 위한 웹 서비스를 개발하는 팀입니다. 다양한 오픈 API를 활용하여 편리하고 유용한 여행 정보를 제공하고자 합니다. 반려동물과 함께하는 여행이 더욱 즐겁고 편리하도록 도와주는 서비스를 만들고 있습니다!
